Designed and simulated a reconfigurable all-optical digital logic system integrating XOR, AND, OR gates and a 2×1 multiplexer using Cross-Phase Modulation (XPM) in Highly Nonlinear Fiber (HNLF). The system operates entirely in the optical domain, eliminating optical–electrical–optical (O-E-O) conversion to achieve high-speed (10 Gb/s), low-latency, and energy-efficient signal processing. By exploiting nonlinear interaction between high-power pump and low-power probe signals, controlled wavelength shifts were generated and selectively filtered using Band-Pass Filters (BPFs) to realize multiple logic operations and multiplexing. Simulation results in OptiSystem demonstrate clear logical outputs with strong extinction ratios, highlighting the potential of photonic logic for scalable optical computing and reversible information processing.
